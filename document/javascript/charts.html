图表组件

概述用法

给指定的元素初始化图表，其中要传入填充的数据，暂时只支持线形图。


图表dom元素必须定义宽高


                    
                        var myChart = new UICharts('j-chart');
                        

                        myChart.showLoading({effect: 'whirling'});
                        

                        myChart.setOption(data);

参数配置

图片对象的创建需要传入dom对象或dom对象的id。提供了showLoading、hideLoading和setOption接口。

属性 默认值 描述
id|object new UIChart(id|object) 创建一个图表
showLoading {text: 'Loading...', effect: 'spin' | 'bar' | 'ring' | 'whirling' | 'dynamicLine' | 'bubble'} 显示loading效果
hideLoading 无 隐藏加载层
setOption {label: [], series: []} label为x坐标文字，series为折线数据,每项的data长度应与label长度一致 
var data = {
        //X坐标
        labels : ["03-01", "03-02", "2/30", "2/30", "2/30", "2/30", "2/30", "2/30", "2/30", "2/30", "2/30", "2/30"],
        //利率数据
        series : [
        // 第一条线
        {
            name:'本月（3月）',
            data:["200", "900", "200", "700", "200", "200","100","100","100","100","100","100"]
        },
        // 第二条线
        {
            name:'上月（2月）',
            data:["100", "700", "900", "800", "600", "400","300","150","180","100","200","100"]
        }
    ]
    };


    var myChart = new UICharts('j-chart');
    myChart.showLoading({effect: 'whirling'});
    myChart.setOption(data);
